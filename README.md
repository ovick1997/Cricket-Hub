# CricketHub — Live Cricket Scoring Platform

Multi-tenant cricket match management, live scoring, and tournament organization platform.

## Features

- 🏏 Live ball-by-ball scoring with real-time updates
- 📊 Detailed scorecards, batting & bowling statistics
- 🏆 Tournament management (League, Knockout, Round-Robin, Short Chris)
- 👥 Team & player management with role-based access
- 📱 Mobile-first responsive design (PWA ready)
- 🔔 Real-time notifications
- 📈 Analytics dashboard with charts
- 🌐 Public pages for live matches, results & leaderboard

## Tech Stack

- **Frontend:** React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend:** Supabase (PostgreSQL, Auth, Edge Functions, Realtime)
- **Animations:** Framer Motion
- **Charts:** Recharts

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
